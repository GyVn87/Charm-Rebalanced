A mod that balances charms!

This is my first time modding Hollow Knight, there are so much to learn so the progress speed is not really good :/

| Charm  | Change | Reason |
| ------------- | ------------- | ------------- |
| Soul Catcher  | Soul gain when Soul Meter is full: 2 => 3  | This makes Soul Catcher more viable in late game with the help of Soul Vessels without making it too strong in early game. |
| Soul Eater  | Notch cost: 4 => 3 <br> Soul gain when Soul Meter is not full: 8 => 7  | The 4-notch cost is extremely high and I don't find the benefits really worth it. Hopefully this change makes Soul Eater more efficient to use.  |
| Grubsong  | Soul gain now SCALES with the amount of damage taken <br> Soul gain when taking damage: 15 => 12 <br> Soul gain when having Grubberfly's Elegy: 25 => 22  | Grubsong is a little too good for early game. So I think it is fine to nerf it a little bit. However, by scaling soul gain with damage, it remains viable in later stages, especially againsts bosses that deal more than 1 damage.
